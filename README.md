# Advent of Code Solutions in Tomo

My solutions to [Advent of Code](https://adventofcode.com/) puzzles implemented in the [Tomo programming language](https://github.com/bruce-hill/tomo).

## Structure

```
advent-of-code-tomo/
├── 2024/
│   ├── day01.tm
│   ├── day02.tm
│   └── ...
├── 2025/
│   └── ...
├── utils/
│   └── helpers.tm (shared utility functions)
└── README.md
```

## Running Solutions

```bash
tomo 2024/day01.tm
```

## About Tomo

Tomo is a modern, statically-typed programming language that compiles to C. Features include:
- Pattern matching
- Optional types
- Array comprehensions
- String interpolation
- Path literals for file I/O
